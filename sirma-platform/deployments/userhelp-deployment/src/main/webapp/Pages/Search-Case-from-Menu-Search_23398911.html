Sirma Platform : Search Case from Menu Search

The Search operation allows the user to search among all cases or other system objects to which he/ she has access.

  • The user starts the search operation from his/ her Personal Dashboard/ Search (1-2)
  • The Basic Search screen opens (3) and the user selects from the drop-down menu Object Type - Case (4-5).

  • The user selects other search criteria if needed (1-2). The following are available: Object type, Object relationship, Search context, Date Created From, Date Created To, Created By.
  • The user clicks the button "Search" (3)
  • The system displays the search results in a table view, paginated (4-5), which contains the following columns: Case Unique Identifier, Case Type, Case State, Created by, date and time of creation. The Case Unique Identifier, Case Type, Case State are a direct link to open the corresponding case.
  • The results may be sorted by different criteria (6), in ascending or descending order (7).

Note: See also topic: Search Options/ Basic Search
